# アプリ仕様書

## 1. 概要
複数のミニアプリを1つのポータルに集約したコレクション型アプリです。トップのダッシュボードから各機能へ遷移し、日常の記録やタスク管理、クイズ、旅行管理を提供します。

## 2. 技術スタック
- Frontend: Vite + React
- 言語: TypeScript
- ルーティング: React Router（BrowserRouter）
- スタイリング: Tailwind CSS
- UI: shadcn/ui（Radix UI + Lucide React）

## 3. 画面構成とルート
- / : ダッシュボード（アプリ一覧）
- /night-routine : ナイトルーティーン
- /todo : ホーム・マネジメント（ToDo）
- /diary : 一言日記
- /quiz : クイズトップ
- /quiz/play : クイズ出題
- /quiz/result : クイズ結果
- /travel : 旅綴（旅行管理トップ）
- /travel/packing : パッキング管理

## 4. 共通UI
- 共通レイアウト（ヘッダー/背景）を適用
- トップ以外の画面で「ホーム」ボタンを表示
- 画面遷移時にスクロール位置をリセット

## 5. 機能仕様

### 5.1 ダッシュボード
- 各アプリへのカードリンクを表示
- カードにはタイトル/説明/状態を表示

### 5.2 ナイトルーティーン
- 就寝前タスクのチェックリスト
- チェック時に完了日時を記録
- 18時以降に当日未リセットなら自動リセット
- ローカルストレージに保存
- タスク追加・削除、編集モード、確認モーダル

### 5.3 一言日記
- 月次カレンダーで日付を表示
- 日付を選択して日記を登録/編集
- 未来日は入力不可
- ローカルストレージに保存

### 5.4 ホーム・マネジメント（ToDo）
- タスクの追加/完了/削除
- 期限入力（任意）
- 完了タスクは7日後に自動削除
- 期限当日の未完了タスクを通知（許可時）
- ローカルストレージに保存

### 5.5 クイズ
- トップ画面から開始
- 出題画面で選択肢を選択し回答を確定
- 全問終了で結果画面へ遷移
- 結果画面にスコアを表示
- リトライで1問目から再開
- 満点のときのみクラッカー演出
- カーテンコール演出（1.2秒）

### 5.6 旅綴
- 旅行管理の入口ページ
- パッキング管理へのリンクを表示

### 5.7 パッキング管理
- カテゴリを作成して持ち物を管理
- カテゴリの開閉（1つだけ展開）
- アイテムの追加
- 編集モードでカテゴリ削除が可能

## 6. データ保存
- ローカルストレージ
  - ナイトルーティーン
  - 一言日記
  - ToDo
- クイズ/旅行は画面状態で管理（セッション内）

## 7. 非機能要件
- レスポンシブ対応（最大幅で中央寄せ）
- 画面遷移やカードに軽いアニメーション
- アクセシビリティ配慮（ボタン/入力の明確な操作）
